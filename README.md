# Análise de Sentimentos dos Tweets Relacionados ao Superior Tribunal Federal

Um projeto desenvolvido em R para o trabalho de conclusão de curso do Bacharelado em Sistemas de Infomação da Universidade Federal de Pernambuco.

## Detalhes

Os algoritmos foram desenvolvidos com o objetivo de executar as seguintes etapas do processo de análise de sentimentos do trabalho:

- Coleta de dados.
- Pré-processamento.
- Classificação.
- Análise e geração de gráficos.
